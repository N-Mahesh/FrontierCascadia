# Frontier Cascadia Website

Marketing site for **Frontier Cascadia**, a Seattle high school hackathon.

## Tech Stack

- Vite
- Vanilla JavaScript
- GSAP
- HTML/CSS

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in your terminal (usually `http://localhost:5173`).

## Available Scripts

- `npm run dev` - Start local dev server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## Project Structure

- `index.html` - Main page markup
- `style.css` - Main site styles
- `main.js` - Main site behavior and interactions
- `public/` - Static assets (icons, favicon, etc.)
- `src/` - Starter Vite scaffold files (not used by the main landing page)

## Deployment

This project is configured for Netlify forms and can be deployed to Netlify as a static site.
